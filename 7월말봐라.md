3. **상대순위 채점** — 모집단별 Top1 winner, Top3 안 winner, Spearman, NDCG를 보고한다. 단일후보 날을 Top1 분모에 섞지 않는다. 동률 winner 처리도 명시한다.

5. **75 vs 80·상단컷·재료·경보 가설** — 기존 가설은 forward에서 별도 코호트로 비교한다. 표본이 작다는 이유로 행을 삭제하지 말고 n과 종목명을 같이 제시한다.

### 판정 상태

| 값 | 의미 | 선행률 분모 |
|---|---|---|
| `plus7_first` | +7%가 먼저 발생 | 포함(승) |
| `minus5_first` | -5%가 먼저 발생 | 포함(패) |
| `same_minute_unknown` | 동일 최초 1분봉에서 양쪽 발생 | 제외·건수 공개 |
| `neither` | 정규장 종료까지 둘 다 미도달 | 제외·건수 공개 |
| `minute_missing` | 분봉 없음 | 제외·원인 확인 |
| `trade_date_mismatch` | 기대 결과일과 API 분봉일 불일치 | 제외 |
| `entry_source_conflict` | 확정 KRX 종가 확인 실패 | 제외 |
| `no_trade` | 결과일 해당 종목 일봉 없음 | 제외 |

판정은 각 분봉의 시가를 그 분봉의 고가·저가보다 먼저 처리한다. 따라서 시가 갭뿐 아니라 VI·거래정지 재개 뒤 첫 체결이 임계값을 건넌 경우도 알려진 순서를 보존한다. 시가가 임계값 안쪽이고 같은 분봉의 고가·저가가 양쪽 임계값을 모두 넘으면 임의로 순서를 만들지 않는다.
